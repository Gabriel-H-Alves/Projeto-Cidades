## 📌 Sobre o Projeto

O **Cidades Tecnológicas de São Paulo** é um site informativo desenvolvido durante o curso **PROA**, como parte das atividades práticas de aprendizagem em desenvolvimento web.

O projeto tem como foco a apresentação de informações organizadas sobre cidades com relevância tecnológica no estado de São Paulo, utilizando uma interface simples, clara e acessível.

---

## 🎓 Contexto Acadêmico

Este projeto foi desenvolvido no programa **PROA**, uma iniciativa voltada à formação de jovens para o mercado de tecnologia, com ênfase em:

* Desenvolvimento web
* Lógica de programação
* Boas práticas de código
* Preparação para o mercado de trabalho

Desenvolver um site simples, funcional e responsivo para:

* Melhorar organização de conteúdo
* Desenvolver senso de design e usabilidade

## 🚀 Como Executar

1. Clone o repositório:

```
git clone https://github.com/Gabriel-H-Alves/Projeto-Cidades.git
```

2. Acesse a pasta do projeto:

```
cd Projeto-Cidades
```

3. Abra o arquivo `index.html` no navegador
